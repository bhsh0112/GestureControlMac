## 0 GetStart

python>=3.8

### 0.1 安装pytorch

[pytorch官网](https://pytorch.org/)查找并安装与个人设备适配的pytorch版本

### 0.2 安装其他软件环境

运行：

```
pip install -r requirements.txt
```

### 0.3 快捷键设置

​		用户可以通过修改`shortcut.json`中的内容，指明每一手势对应的快捷键，完成对快捷键的设置（支持的手势与快捷键见下表，可以对两个表格的内容进行排列组合设置）

##
